Non-Contact Pointing Device

Sometimes, it is not feasible to control a computer by touching a device such as a mouse. For example, in cleanroom environments, sterility requirements may make a mouse impractical. This software controls a computer using a metal rod shown below. To move the mouse cursor, the user moves the tip of the rod.

Features

  • Mouse click simulated by pressing a foot pedal
  • Process executes a small TensorFlow Object Recognition model (AutoML)
  • Process runs asynchronously to avoid cursor freezing if calling program blocks execution

Technologies used

  • Python
    • OpenCV
    • Multiprocessing
  • TensorFlow/AutoML
